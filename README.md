# 1st RGP Hackathon Festival
제 1회 RGPKorea 해커톤 - 인스타그램 API 를 활용한 아이폰 앱 만들기

## 프로젝트명: IngsMuk (가칭)

### 목표
휴대폰의 GPS 위치정보를 이용하여, 주변 근거리의 인스타그램 미디어들을 검색하고, 그 중 "#맛집" 태그를 가진 미디어만 필터링 하여 보여준다.
